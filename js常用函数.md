* 常规函数
```
	(1)alert	显示一个警告对话框，包括一个OK按钮。
	(2)confirm	显示一个确认对话框，包括OK、Cancel按钮。
	(3)escape	将字符转换成Unicode码。
	(4)eval	计算表达式的结果。
	(5)isNaN	测试是(true)否(false)不是一个数字
	(6)parseFloat	将字符串转换成符点数字形式。
	(7)parseInt	将符串转换成整数数字形式(可指定几进制)。
	(8)prompt	显示一个输入对话框，提示等待用户输入。
	(9)unescape	解码由escape函数编码的字符。
```
* Array对象数组函数		
```
	(1)join	设置分隔符连接数组元素为一个字符串。
	(2)langth	返回数组的长度。
	(3)reverse	将数组元素顺序颠倒。
	(4)sort	将数组元素重新排序
	(5)concat	组合数组
	(6)slice(开始下标，结束下标)	截取数组
	(7)splice(开始下标，结束下标)	
	(8)push	尾部插入数据
	(9)pop	尾部删除数据
	(10)unshift	开头插入数据
	(11)shift	开头删除数据
```
* Data对象日期函数		
```
	(1)getDate	返回日期的“日”部分，值为1～31。
	(2)getDay	返回星期，值为0～6，0表示星期日。
	(3)getHours	返回日期的“小时”部分，值为0～23。
	(4)getMinutes	返回日期的“分钟”部分，值为0～59。
	(5)getMonth	返回日期的“月”部分，值为0～11
	(6)getSeconds	返回日期的“秒”部分，值为0～59。
	(7)getTime	返回系统时间。
	(8)getTimezoneOffset	返回此地区的时差(当地时间与GMT格林威治标准时间的地区时差)，单位为分钟。
	(9)getYear	返回日期的“年”部分。返回值以1900年为基数，如1999年为99。
	(10)parse	返回从1970年1月1日零时整算起的毫秒数(当地时间)。
	(11)setDate	设定日期的“日”部分，值为0～31。
	(12)setHours	设定日期的“小时”部分，值为0～23。
	(13)setMinutes	设定日期的“分钟”部分，值为0～59。
	(14)setMonth	设定日期的“月”部分，值为0～11。其中0表示1月，...，11表示12月。
	(15)setSeconds	设定日期的“秒”部分，值为0～59。
	(16)setTime	设定时间。时间数值为1970年1月1日零时整算起的毫秒数。
	(17)setYear	设定日期的“年”部分。
	(18)toGMTString	转换日期成为字符串，为GMT格林威治标准时间。
	(19)setLocaleString	转换日期成为字符串，为当地时间。
	(20)UTC	返回从1970年1月1日零时整算起的毫秒数(GMT)。
```
